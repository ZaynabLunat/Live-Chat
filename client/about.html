Why Web Sockets?

  • Two way communication between the client and server
  • Interactive communication between any number of clients and the server
  • Requires less bandwidth and has less latency compared to HTTP
  • Useful when you need updates in real time without having waiting too long
  • Great for: live chats, online games, and GPS
  • Enhances user experience
